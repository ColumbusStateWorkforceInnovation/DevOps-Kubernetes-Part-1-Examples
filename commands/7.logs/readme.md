
### `kubectl logs`
`kubectl logs` outputs `stdout` and `stderr` logs from a pod. If more than one container exists use the `-c` flag with the container name.

**Command**
```
kubectl logs <NAME>
kubectl logs <NAME> -c <container-name>
```

**Examples**
```
$ kubectl logs basic-pod
```
